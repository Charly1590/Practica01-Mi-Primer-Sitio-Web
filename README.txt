Para comenzar con este sitio web se utilizo una extensión del Visual Studio code llamada Live Server la cual nos creara una carpeta con un archivo json que nos permitirá ver los cambios que hagamos en tiempo real desde el cliente web
Una página principal denominada index.html
Para el desarrollo de la pagina principal index.html se realizará la estructura básica del HTML5 el mismo que se utilizara para las 8 paginas .html restantes.
Como podemos ver el documento cuenta con un atributo UTF-8 el cual nos permite tener una codificación de caracteres para tener un formato UNICODE, también contamos con el
name="viewport" content="width=device-width, initial-scale=1.0" el cual nos ayudara a que nuestras paginas web puedan verse de manera correcta en dispositivos móviles.
Ocho páginas en formato *.html secundarias con temas relacionados a la página principal. Nota.- las páginas html, imágenes, audios, videos y otros recursos deberán estar organizadas en carpetas.
Todas las páginas deberán tener un menú de navegación el cuál, permite al usuario navegar de una manera clara y sencilla entre todas las páginas HTML, tanto la principal como la secundaria
Como podemos ver el menú de navegación esta dentro de un header que se colocó al principio del body para que este al principio de la página y que el usuario pueda encontrarla fácilmente.
La navegación esta organizada en una lista no ordenada representada por puntos y de manera vertical.
Este modelo de navegación es el que se utilizara en las ocho paginas restantes.
Todas las páginas *.html deben estar estructuradas según la semántica de HTML5. Nota.- se debe explicar la estructura de cada una de las páginas:
